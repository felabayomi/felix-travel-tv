Step 1 – Content Type Classification Logic

Your AI must first determine what type of content it received.

Classification Logic

IF input contains city + activities → City Discovery
IF input contains destination description → Destination Feature
IF input contains pricing + dates → Travel Deal / Trip
IF input contains app/software description → Travel Tool / App
IF input contains route/stops → Road Trip
IF input contains tips only → Travel Tips

So first step is always:

content_type = classify(input)


⸻

Extraction Logic

So:

data = extract_information(input)

chapters = [

Step 4 – Content Expansion Logic

Now the AI expands each chapter into content.

Expansion Logic

For each chapter:

function generate_travel_content(input):

    content_type = classify(input)

    data = extract_information(input)

    chapters = build_chapters(data)
